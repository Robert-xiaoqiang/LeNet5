# LeNet5 in TensorFlow
##  prerequisite
   - Python >= 3.5.1
   - TensorFlow@CPU
   - NumPy

## directory
   - `data/*` 4 `.gz` files(training and testing data set from`http://yann.lecun.com/exdb/mnist/`)

## train
   - `Python YetLeNet.py`

## note
   - copy those 4 `.gz` files to `/data/` before training

## a naive desktop application
   - `Python YetTestIt.py`
